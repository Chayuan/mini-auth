# Mini-auth

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Description 

This repo is not production ready. It is a simple file-system base JWT auth server based on NestJS.
It works natively with <a href="https://native-ship.dev">NativeShip</a>
